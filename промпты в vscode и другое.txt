Проверка наличия всех удаляемых UWP-приложений, которые не являются фреймворками:

Под строку 230 размести раскрывающийся элемент: Отключение Recall / Windows AI
Внутри следующий текст:
В Windows есть функция Recall, которая каждые несколько секунд делает скриншоты экрана и использует искусственный интеллект для анализа и сортировки информации. Настоятельно рекомендуется отключить его.

Для отключения Recall выполните в Powershell от имени Администратора команду:
DISM /Online /Disable-Feature /Featurename:Recall  (кнопка "Копировать")

Для проверки состояния Recall выполните в Powershell от имени Администратора команду:
DISM /Online /Get-Featureinfo /Featurename:Recall  (кнопка "Копировать")

Редактор локальной групповой политики:
В командной строке выполните команду:
gpedit.msc   (кнопка "Копировать")

В открывшемся Редакторе групповой политики перейдите по следующему пути: Конфигурация пользователя -> Административные шаблоны -> Компоненты Windows -> Windows Copilot
Перейдите в изменение параметров политики "Отключить Windows Copilot". Выберите парметр "Включено" и нажмите "Применить". Обратите внимание на то, что по умолчанию при ненастроенной политике Windows может делать скриншоты.
